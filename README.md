# Notebook

A quiet corner for scattered thoughts and structured knowledge

## 💡 創作原由 | Rationale
這是一個因為「想做筆記」而誕生的空間。在學習的過程中，我希望能有一個屬於自己的知識庫。剛好近期正在實踐 **Vibe Coding**，我認為不該只是單純的記錄文字，而是可以利用科技與創意的結合，透過程式碼與設計美學，為枯燥的學科知識建立一個兼具視覺享受與功能性的筆記庫。

This space was born from a simple desire to take notes. Throughout my learning journey, I wanted a personal knowledge base. While practicing **Vibe Coding**, I realized that note-taking shouldn't just be about text. By combining technology and creativity—through code and design aesthetics—I aim to transform academic subjects into a visually engaging and functional digital library.
